# 🏪 FranManager
> JAVA 콘솔을 이용한 가맹점 관리 시스템  
> 본사 차원의 제품 및 가맹점 데이터 관리 + 통계 기반 마케팅 전략 수립 프로젝트

---

## 📆 프로젝트 기간
**2025.07.28 ~ 2025.08.08 (2주)**

---

## 🎯 프로젝트 개요
**FranManager**는 Java 콘솔 환경에서 가맹점 및 제품을 효율적으로 관리할 수 있도록 설계된 시스템입니다.  
기본적인 CRUD 기능 외에도 통계 데이터를 활용한 마케팅 전략 구축이 가능하도록 구현되었습니다.

---

## ⚙️ 개발 목표
- JAVA 콘솔 기반 가맹점 관리 시스템 구현  
- 다양한 통계 자료를 통한 효율적인 마케팅 전략 수립  
- Git을 활용한 버전 관리 및 협업 프로세스 경험

---

## 🧩 주요 기능
- ✅ 가맹점 정보 등록 / 수정 / 삭제 / 조회 (CRUD)
- ✅ 제품 및 발주 내역 관리
- ✅ SQL 통계 기능 (판매량, 수익률 등)
- ✅ 통계 기반 마케팅 전략 분석
- ✅ Git을 통한 버전 관리 및 협업 이력 관리

---
