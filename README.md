## 🧭 프로젝트 개요

**FranManager**는 Java 콘솔 환경에서 가맹점 및 제품을 효율적으로 관리할 수 있도록 설계된 시스템입니다.
기본적인 CRUD 기능 외에도 통계 데이터를 활용한 마케팅 전략 구축이 가능하도록 구현되었습니다.

- **인원:** 3명

  - **최규호** – 가맹점·판매현황 관

## 🧩 주요 기능

### 👩‍💼 본사 관리 기능
- **1. 🏪 가맹점 관리**
  - 가맹점 리스트 조회, 추가, 정보 수정, 삭제 (CRUD)
- **2. 🍵 제품 관리**
  - 제품 전체 조회, 등록, 수정, 판매상태 변경
- **3. 📦 입출고 관리**
  - 재고 현황 보기, 입고 등록, 입출고 로그 조회, 입출고 내역 수정
- **4. 🚚 발주 관리**
  - 가맹점 발주 요청 보기, 출고 처리, 발주 요청 취소 처리

### 📊 데이터 분석 기능
- **5. 💰 판매 현황 보기**
  - 전체 판매 내역 페이지별 조회
- **6. 📈 통계 보기**
  - 제품별, 지역별, 시간대별 판매 통계 조회
- **7. 📝 리뷰 보기**
  - 리뷰 전체 조회, 가맹점별 리뷰 조회, 제품별 리뷰 조회

## 🗃️ 데이터 모델링

- **Fran**: 가맹점 정보 (가맹점명, 주소, 연락처, 점주명, 상태 등)
- **Product**: 제품 정보 (제품명, 공급가, 판매가, 판매상태 등)
- **SupplyLog**: 가맹점 발주(요청) 로그
- **OrderLog**: 가맹점 판매 로그
- **ioLog**: 본사 제품 입출고 로그 (입고=0, 출고=supNo)
- **Review**: 고객 리뷰 (판매번호 기준)
